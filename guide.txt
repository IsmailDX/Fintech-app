- first step we write this command to create the app:
    npx create-expo-app fintech -t tabs

- we install libraries:
    npx expo install expo-dev-client react-native-reanimated react-native-gesture-handler

- create a babel.config file, and add the plugins
    plugins: ['react-native-reanimated/plugin'],

- run the command (this will create an android and ios folder in your project):
    npx expo prebuild

- run the command (to run app, Might take time only the first time):
    npx expo run:android --device

- if everything is working, close the server and run (asset used to load the video files, and asset is for playing the video):
    npx expo install expo-asset
    npx expo install expo-av
