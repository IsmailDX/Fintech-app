==================================================
             EXPO APP WORKFLOW GUIDE
==================================================

--------------------------------------------------
üÜï PART A: FIRST-TIME SETUP (New Projects Only)
--------------------------------------------------
1. Create App: 
   npx create-expo-app fintech -t tabs

2. Install Core Native Libs:
   npx expo install expo-dev-client react-native-reanimated react-native-gesture-handler expo-video

3. Configure Babel:
   Add 'react-native-reanimated/plugin' to the END of your plugins in babel.config.js:
   plugins: ['react-native-reanimated/plugin']

4. Generate Native Folders:
   npx expo prebuild

5. Run:
   npm run android

--------------------------------------------------
üîÑ PART B: SYNCING DEVICES (After Git Pull)
--------------------------------------------------
Use this routine every time you switch laptops:

1. Sync JS:     npm install
2. Sync Native: npx expo prebuild --clean
3. Run:         npm run android

--------------------------------------------------

- You switched laptops or changed NDK/SDK versions
- You get "Native Module not found" errors
